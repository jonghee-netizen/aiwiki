# LLM (대규모 언어 모델)

**Large Language Model(LLM)**은 방대한 양의 텍스트 데이터로 사전 훈련된 신경망 기반의 언어 모델입니다. 자연어 이해, 생성, 번역, 요약 등 다양한 언어 작업을 수행할 수 있습니다.

## 핵심 개념

### 트랜스포머(Transformer)
LLM의 근간이 되는 신경망 아키텍처입니다. 2017년 "Attention is All You Need" 논문에서 처음 제안되었습니다.

### 토큰(Token)
LLM이 텍스트를 처리하는 기본 단위입니다. 단어, 단어의 일부, 또는 문장 부호가 될 수 있습니다.

```
"안녕하세요" → ["안녕", "하세요"] (약 2-3 토큰)
"Hello world" → ["Hello", " world"] (2 토큰)
```

### 컨텍스트 윈도우
모델이 한 번에 처리할 수 있는 최대 토큰 수입니다.

| 모델 | 컨텍스트 윈도우 |
|------|----------------|
| GPT-4o | 128,000 토큰 |
| Claude 3.x | 200,000 토큰 |
| Gemini 1.5 Pro | 1,000,000 토큰 |

## 작동 원리

1. **사전 훈련(Pre-training)**: 대량의 텍스트로 다음 토큰 예측 학습
2. **미세 조정(Fine-tuning)**: 특정 작업에 맞게 추가 훈련
3. **RLHF**: 인간 피드백으로 강화학습하여 응답 품질 향상

## 주요 능력

- 텍스트 생성 및 완성
- 질의응답
- 코드 생성 및 분석
- 번역 및 요약
- 감정 분석
- 분류 및 추출

## 한계

- **환각(Hallucination)**: 사실이 아닌 내용을 자신있게 생성
- **지식 컷오프**: 훈련 데이터 이후의 정보는 모름
- **추론 한계**: 복잡한 수학적 추론에서 오류 발생 가능
- **편향성**: 훈련 데이터의 편향이 모델에 반영될 수 있음
