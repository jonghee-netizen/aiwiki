# 메인 페이지 브랜드 설정 (나스 소재 찾기)

기본 필터 키워드: `[FB]` (활성화됨)

| 브랜드 표시명 | config.json name | NAS 경로 | 인덱싱 폴더 (year_filter) |
|-------------|-----------------|---------|--------------------------|
| 샤르드 | `01 샤르드` | `Z:\01 샤르드\2. 업로드` | 전체 |
| 트리코닉스 | `02 트리코닉스` | `Z:\02 트리코닉스\2. 업로드` | 전체 |
| 닥터빼 | `03 닥터빼` | `Z:\03 닥터빼\2.업로드` | 여유컷패치, 피치스팟패치 |
| 이옴 | `05 이옴` | `Z:\05 이옴\03. 업로드\01. 트러블 패치 마스크` | 2025년도, 2026년도 |
| 링탭 | `06 링탭` | `Z:\06 링탭\업로드` | 굿나잇패치, 글루타치온패치 |
| 솔티스 | `솔티스` | `Y:\광고팀\1_솔티스\2. 업로드` | 눈, 타마플렉스, 클리어세트, 오메가, 라피노스, 기관지, 전립선 |
| 화이니카 | `화이니카` | `X:\광고팀\화이니카\2.업로드` | 치탄플러스 |

## 동작 방식

- 브랜드 탭 클릭 → 해당 `year_filter` 폴더만 NAS 스캔
- `year_filter` 없는 브랜드는 경로 전체 스캔
- 결과는 `index_cache.json`에 저장
